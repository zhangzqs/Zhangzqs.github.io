P1824-进击的奶牛

发表于2022-01-21|更新于2022-01-21
|字数总计:677|阅读时长:2分钟|阅读量:|评论数:

题目描述

Farmer John 建造了一个有 N$(2 ≤ N ≤ 100000) $个隔间的牛棚，这些隔间分布在一条直线上，坐标是 x1,...,xN(0≤xi≤1000000000)x_1 ,...,x_N (0 ≤ x_i ≤ 1000000000)x1​,...,xN​(0≤xi​≤1000000000)。

他的 $C(2 ≤ C ≤ N) $头牛不满于隔间的位置分布，它们为牛棚里其他的牛的存在而愤怒。为了防止牛之间的互相打斗，Farmer John 想把这些牛安置在指定的隔间，所有牛中相邻两头的最近距离越大越好。那么，这个最大的最近距离是多少呢？

输入格式

第 1 行：两个用空格隔开的数字 N和 C。

第 2 ~ N+1 行：每行一个整数，表示每个隔间的坐标。

输出格式

输出只有一行，即相邻两头牛最大的最近距离。
