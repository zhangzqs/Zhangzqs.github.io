  • жџЈТю║тЈўжЄЈ $ X $№╝џС╗БУАеТйютюетђ╝уџётѕєтИЃсђѓ
  • ТдѓујЄт»єт║дтЄйТЋ░№╝ѕPDF№╝Ѕ$ X \sim p(x) $№╝џТЈЈУ┐░жџЈТю║У┐ЄуеІжђЅтЈќтђ╝ $ x $ уџёуЏИт»╣ТдѓујЄсђѓ
  • уц║СЙІ№╝џтЮЄтїђуџёТдѓујЄт»єт║дтЄйТЋ░№╝џтюеСИђСИфт«џС╣ЅтЪЪтєЁТЅђТюЅтђ╝уџётЈ»УЃйТђДтЮЄуГЅсђѓ
    • СЙІтдѓ№╝џтЁГжЮбжф░тГљ№╝ї$ X $ тЈќтђ╝СИ║ $ 1,2,3,4,5,6 $№╝їСИћ $ p(1)=p(2)=p(3)=p(4)=p(5)=p(6) $сђѓ

ТдѓујЄ Probabilities

  • тГўтюе $ n $ СИфуд╗ТЋБтђ╝ $ x_i $№╝їТ»ЈСИфтђ╝т»╣т║ћТдѓујЄ $ p_i $сђѓ
  • ТдѓујЄтѕєтИЃуџёУдЂТ▒ѓ№╝џ
    • $ p_i \geq 0 $
    • $ \sum_{i = 1}^{n} p_i = 1 $
  • тЁГжЮбжф░тГљуц║СЙІ№╝џ$ p_i = \frac{1}{6} $

жџЈТю║тЈўжЄЈуџёТюЪТюЏ Expected Value of a Random Variable

  • Тў»С╗јжџЈТю║тѕєтИЃСИГтЈЇтцЇТійтЈќТаиТюгТЌХтЙЌтѕ░уџёт╣│тЮЄтђ╝сђѓ
  • УІЦжџЈТю║тЈўжЄЈ $ X $ ТЮЦУЄфтЁиТюЅ $ n $ СИфуд╗ТЋБтђ╝ $ xi $№╝ѕт»╣т║ћТдѓујЄСИ║ $ p_i №╝ЅуџётѕєтИЃ№╝ї№╝ЅуџётѕєтИЃ№╝ї№╝ЅуџётѕєтИЃ№╝ї X $ уџёТюЪТюЏтђ╝СИ║№╝џ$ E[X] = \sum{i = 1}^{n} x_i p_i $сђѓ
  • жф░тГљуц║СЙІ№╝џ$ E[X] = \sum_{i = 1}^{6} \frac{i}{6} = (1 + 2 + 3 + 4 + 5 + 6)/6 = 3.5 $сђѓ

ТдѓујЄт»єт║дтЄйТЋ░ Probability Density Function (PDF)

  • жџЈТю║тЈўжЄЈ $ X $ Т╗АУХ│ $ X \sim p(x) $№╝їтЈ»тЈќУ┐ъу╗ГУїЃтЏ┤тєЁуџёС╗╗ТёЈтђ╝№╝їТЪљСИђуЅ╣т«џтђ╝уџёуЏИт»╣ТдѓујЄућ▒У┐ъу╗ГТдѓујЄт»єт║дтЄйТЋ░ $ p(x) $ у╗ЎтЄ║сђѓ
  • $ p(x) $ жюђТ╗АУХ│уџёТЮАС╗Х№╝џ$ p(x) \geq 0 $ СИћ $ \int p(x) dx = 1 $сђѓ
  • $ X $ уџёТюЪТюЏтђ╝№╝џ$ E[X] = \int x p(x) dx $сђѓ

жџЈТю║тЈўжЄЈуџётЄйТЋ░ Function of a Random Variable

  • жџЈТю║тЈўжЄЈ $ X $ уџётЄйТЋ░ $ Y $ С╣ЪТў»СИђСИфжџЈТю║тЈўжЄЈ№╝џ
    • УІЦ $ X \sim p(x) $№╝їСИћ $ Y = f(X) $сђѓ
  • жџЈТю║тЈўжЄЈтЄйТЋ░уџёТюЪТюЏтђ╝№╝џ
    $ E[Y] = E[f(X)] = \int f(x) p(x) dx $
