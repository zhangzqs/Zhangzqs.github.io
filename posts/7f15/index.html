用户分享管理 API

创建分享

请求

共享权限

  • 只读权限：“readonly”
  • 读写权限：“readwrite”

POST /share?bucket=xxx

Authorization: <普通用户鉴权>

发送延时消息，同时订阅消息，时间到后自动调用删除分享

响应

查看分享

GET /share/{:id}

删除分享

DELETE /share/{:id}

Authorization: 普通用户鉴权

列举分享

按照创建时间顺序优先列举

GET /share?bucket=xxx&max-size=xx&marker=xx

Authorization: 普通用户鉴权

对外的终端用户使用的 API

根据分享 id 查询分享信息

GET /public/share/{:id}?token=xxx

根据分享 id 和分享码提取分享信息

GET /public/share/{:id}/extract?token=xxx&extractCode=xxx
